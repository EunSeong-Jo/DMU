# API 키 발급 및 데이터 수집 완벽 가이드

순서대로 따라하면 데이터 수집을 완료할 수 있습니다.

---

## 📌 전체 작업 흐름

```
1단계: API 키 1개 발급 (10분) - 공공데이터포털만!
   ↓
2단계: API 키를 코드에 입력 (2분)
   ↓
3단계: 자동 데이터 수집 실행 (1-2시간) - 아파트 실거래가만
   ↓
4단계: 수동 데이터 다운로드 (30분) - 나머지 모든 데이터
   ↓
완료!
```

---

## 1단계: API 키 발급하기

**실제로는 1개의 API 키만 발급받으면 됩니다!** (공공데이터포털)

### 🔑 API 키 #1: 국토교통부 (아파트 실거래가)

**Step 1) 공공데이터포털 회원가입**
1. 브라우저 주소창에 입력: `www.data.go.kr`
2. 우측 상단 "회원가입" 클릭
3. 회원가입 완료 후 로그인

**Step 2) API 신청하기**
1. 로그인 상태에서 검색창에 "아파트매매 실거래" 입력
2. 검색 결과에서 "아파트매매 실거래 상세 자료" 클릭
3. 페이지 중간의 파란색 "활용신청" 버튼 클릭
4. 활용목적 입력란에 작성: "빅데이터 분석 학습 프로젝트"
5. 상세기능정보 체크박스 모두 선택
6. "활용신청" 버튼 클릭 → 즉시 승인됨

**Step 3) API 키 확인 및 복사**
1. 우측 상단 "마이페이지" 클릭
2. 좌측 메뉴에서 "오픈API" 클릭
3. "개발계정" 탭 클릭
4. "일반 인증키 (Encoding)" 부분의 긴 문자열 전체 복사
5. 메모장에 저장:
   ```
   국토교통부 API 키: [여기에 붙여넣기]
   ```

**이게 끝입니다!**

나머지 데이터(지하철, 버스, 대기질, 학교 등)는 모두 CSV 파일로 직접 다운로드하면 됩니다.

---

## 2단계: API 키를 코드에 입력하기

메모장에 저장한 API 키를 코드에 입력합니다.

**Step 1) 파일 열기**
1. VSCode에서 프로젝트 폴더 열기
2. `src` 폴더 → `collect_apartment_data.py` 파일 열기

**Step 2) API 키 입력**
1. 파일 맨 아래쪽에 있는 `API_KEY = "YOUR_API_KEY_HERE"` 찾기
2. 따옴표 안에 발급받은 공공데이터포털 API 키 붙여넣기:
   ```python
   API_KEY = "여기에_발급받은_키_붙여넣기"
   ```

**예시**:
```python
API_KEY = "dK8xF3nQ9mB2hV7wP5tL6gR4cN1sM0yX=="
```

3. `Ctrl + S`로 저장

---

## 3단계: 아파트 실거래가 데이터 수집 (자동)

API를 사용하여 아파트 실거래가만 자동으로 수집합니다.

**Step 1) 터미널 열기**
1. VSCode 상단 메뉴 → Terminal → New Terminal
2. 또는 단축키: `Ctrl + \` (백틱)

**Step 2) 실행 위치로 이동**
터미널에 다음 명령어 입력:
```bash
cd src
```

**Step 3) 아파트 실거래가 수집 실행**
터미널에 다음 명령어 입력:
```bash
python collect_apartment_data.py
```

**Step 4) 진행 상황 확인**
- 화면에 다음과 같은 메시지가 표시됩니다:
  ```
  ==============================================================
  아파트 실거래가 데이터 수집 시작
  기간: 2020년 ~ 2024년
  지역: 서울시 25개 구
  ==============================================================
  ✓ 2020-01 11680: 523건 수집
  ✓ 2020-01 11740: 412건 수집
  ...
  ```

- **예상 소요 시간**: 1-2시간
- **커피 한 잔 하고 오세요!** ☕

**Step 5) 완료 확인**
- 모든 수집이 끝나면:
  ```
  ✅ 데이터 수집이 완료되었습니다!
  📁 저장 위치: data/raw/seoul_apt_transactions_full.csv
  ```

**Step 6) 파일 확인**
`data/raw/` 폴더에 생성된 파일 확인:
- ✅ `seoul_apt_transactions_full.csv` (크기: 약 50-100MB)

## 4단계: 수동 데이터 다운로드

나머지 모든 데이터는 CSV 파일로 직접 다운로드합니다.

### 📥 파일 #1: 지하철역 정보

**Step 1) 사이트 접속**
1. 브라우저 주소창에 입력: `data.seoul.go.kr`
2. 회원가입 없이 바로 다운로드 가능

**Step 2) 데이터 찾기**
1. 검색창에 "지하철역" 입력
2. "서울시 지하철역 정보" 또는 "역별 위치정보" 클릭
3. 페이지에서 "CSV" 또는 "엑셀" 다운로드 버튼 클릭

**Step 3) 파일 저장**
1. 다운로드한 파일 이름을 `subway_stations.csv`로 변경
2. 프로젝트 폴더의 `data/raw/` 폴더에 저장

**필요한 정보**:
- 역명, 호선, 위도, 경도

---

### 📥 파일 #2: 버스정류장 정보

**Step 1) 사이트 접속**
1. `data.seoul.go.kr` 접속

**Step 2) 데이터 찾기**
1. 검색창에 "버스 정류소" 입력
2. "서울시 버스정류소 위치정보" 클릭
3. CSV 다운로드

**Step 3) 파일 저장**
1. 파일명을 `bus_stops.csv`로 변경
2. `data/raw/` 폴더에 저장

**필요한 정보**:
- 정류장명, 위도, 경도

---

### 📥 파일 #3: 대기질 정보

**Step 1) 사이트 접속**
1. 브라우저에 입력: `airkorea.or.kr`
2. 또는 서울시 열린데이터광장에서 "대기질" 검색

**Step 2) 데이터 찾기**
1. "통계자료" 또는 "측정소별 자료" 메뉴
2. 서울시 25개 구 측정소 데이터 다운로드
3. 또는 서울시 열린데이터광장에서 "대기환경정보" CSV 다운로드

**Step 3) 파일 저장**
1. 파일명을 `air_quality.csv`로 변경
2. `data/raw/` 폴더에 저장

**필요한 정보**:
- 구별, PM10 평균, PM2.5 평균

---

### 📥 파일 #5: 범죄 통계

**Step 1) 사이트 접속**
1. 브라우저 주소창에 입력: `www.police.go.kr`

**Step 2) 데이터 찾기**
1. "범죄통계" 메뉴 클릭

3. 서울시 데이터 선택 후 다운로드 (Excel 파일)

**Step 3) 파일 저장**
1. 파일명을 `crime_statistics.xlsx`로 변경
2. `data/raw/` 폴더에 저장

**필요한 정보**:
- 구별, 범죄 발생 건수
