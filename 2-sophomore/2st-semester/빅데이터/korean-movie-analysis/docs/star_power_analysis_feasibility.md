# 스타파워 분석 실현 가능성 평가

## 🚨 현재 상황: 과거 데이터 부족

### 문제점
- **보유 데이터**: 2019-2024년 (5년)
- **필요 데이터**: 최소 2010년 이후 (10~15년)

### 구체적 예시

#### 2019년 영화 "기생충"
```
감독: 봉준호
├─ 필요한 과거 실적 (2019년 이전):
│   - 설국열차 (2013): 930만명 ❌
│   - 마더 (2009): 360만명 ❌
│   - 괴물 (2006): 1,300만명 ❌
│   - 살인의 추억 (2003): 525만명 ❌
└─ 우리 데이터: 없음

주연: 송강호
├─ 필요한 과거 실적:
│   - 택시운전사 (2017): 1,200만명 ❌
│   - 변호인 (2013): 1,137만명 ❌
│   - 관상 (2013): 913만명 ❌
│   - ...약 50편
└─ 우리 데이터: 2019년 이후만
```

**결론**: 2019년 영화의 스타파워를 계산할 수 없음!

## 📊 해결 방안 비교

### **방안 1: 과거 데이터 추가 수집** ⏱️ 2-3시간

#### 수집 범위
- **기간**: 2010-2018년 (9년 추가)
- **예상 영화 수**: 연 상위 50편 × 9년 = 450편
- **총 데이터**: 현재 300편 + 450편 = 750편

#### 수집 방법
```python
# KOBIS API로 2010-2018 박스오피스 수집
for year in range(2010, 2019):
    top_50_movies = get_yearly_boxoffice(year, limit=50)
    # 기본 정보만 수집 (Naver 평점은 불필요)
```

#### 장점
✅ 완전한 스타파워 분석 가능
✅ 감독/배우 경력 곡선 추적 가능
✅ 2019-2024년 영화의 예측 변수로 활용

#### 단점
⏰ 2-3시간 소요 (KOBIS API만, Naver 평점 제외)
📊 분석 복잡도 증가

#### 실현 가능성
⭐⭐⭐⭐⭐ (강력 추천)

### **방안 3: 제한적 스타파워 분석 (현재 데이터만)** ⏱️ 0시간

#### 분석 방법
```python
# 2019-2024년 내에서만 경력 계산
# 예: 2020년 영화의 감독 과거 실적 = 2019년 작품만 고려

for movie in movies_2020:
    director = movie['director']
    past_movies = movies_2019[movies_2019['director'] == director]
    if len(past_movies) > 0:
        star_power = mean(past_movies['audience'])
    else:
        star_power = 0  # 신인 감독
```

#### 장점
⏰ 즉시 실행 가능
✅ 추가 수집 불필요

#### 단점
❌ 대부분 감독/배우가 "과거 실적 없음" 처리됨
❌ 2019년 영화는 분석 불가 (과거 데이터 0)
❌ 스타파워 효과 왜곡 (실제는 대배우인데 데이터상 신인)

#### 실현 가능성
⭐ (비추천 - 의미 없는 분석)

### **방안 4: 스타파워 분석 포기, 다른 변수로 대체** ⏱️ 0시간

#### 대체 변수
1. **경쟁 강도 분석** (현재 데이터로 가능)
   - 같은 주 개봉한 영화들의 총 관객수
   - 강력한 예측 변수

2. **장르 트렌드 분석** (현재 데이터로 가능)
   - 연도별 장르 선호도 변화
   - 장르 포화도 계산

3. **개봉 스크린 수** (KOBIS에 일부 포함)
   - 배급사 투자 규모의 간접 지표

#### 장점
⏰ 즉시 실행 가능
✅ 추가 수집 불필요
✅ 여전히 의미 있는 분석

#### 단점
⚠️ 가장 강력한 변수(스타파워) 누락
⚠️ 예측력 제한적

#### 실현 가능성
⭐⭐⭐⭐ (현실적 대안)
